HOW TO USE
----------

Step 1.
 
Copy decklist into "cards.txt"
if you are using archidekt, uncheck all
Export Options

Step 2.

Run Start.bat

Step 3.

After it fetches all the cards check for
errors (red text). if none are present give a name 
for the document

Step 4. 

Adjust images to your need
